Use for type information, go-to-definition, find-references, symbol listings, and compiler diagnostics when a language server is available. Requires an active LSP backend — returns an error if none is connected. Actions hover, definition, and references require 1-based line and column (matching editor and grep output); symbols and diagnostics need only the file path. Prefer this over grep for type-aware lookups. Do not use for file content — use read instead.
