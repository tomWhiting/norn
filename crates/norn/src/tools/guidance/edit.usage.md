Use for targeted modifications to existing files. Prefer over write when changing a specific section — edit sends only the diff. The old_string must match exactly once in the file; ambiguous or missing matches are rejected. The file must have been read this session. Edits are validated with tree-sitter: syntax errors roll back the change (gate semantics) unless AllowBrokenAst is set. Blast radius (lines added, removed, modified, and containing symbols) is reported in the result.
