Use for locating code, files, or structural patterns. Set mode explicitly: content for regex line matches, files for glob expansion, fuzzy for approximate path lookup, ast for tree-sitter structural queries. When mode is omitted, the tool infers content if pattern is given, files if only glob is given. Use glob to restrict which files are considered in any mode. Use path to scope the search to a specific directory. Results are capped at max_results (default 50) to avoid flooding context. Search is read-only and runs concurrently with other read-only tools.
